# The Start Circuit

## What is this? 
This project is a circuit light that simulates the starting line for races. It has red "get ready", yellow "set", and green is "go". Hope you like it.

## How does this work? 

This circuit is powered by a 9v battery and with 3 leds wired in parallel. The leds stay on after the battery is off using capacitors thart charge up!
